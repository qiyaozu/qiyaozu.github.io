Mar 03, 2020点击

第一步： 安装webbench：

先安装依赖:
brew install ctags

下载并安装：

使用： webbench -c 并发数 -t 运行测试时间 URL
eg： webbench -t 60 -c 100 http://www.baidu.com/

安装Siege：

相关命令：

eg: 1000个人，没人请求100次，测试5秒钟
